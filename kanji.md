## 讀音細分

### 人（じん／にん／ひと）

| 讀音 | 用法 | 例詞 | 說明 |
| --- | ---- | ---- | ---- |
| じん | 國籍・民族後綴 | 日本人・外国人・中国人 | 接國名/地名，表示「…人」 |
| にん | 人數・職業・身分 | 三人（さんにん）・犯人・本人 | 計數或固定複合詞 |
| ひと | 獨立使用 | 人（ひと）・あの人 | 訓読み，指「人」這個存在本身 |

⚠️ 一人（ひとり）・二人（ふたり）是特殊讀法，非 じん 也非 にん。

### 豚（とん／ぶた）

| 讀音 | 用法 | 例詞 | 說明 |
| --- | ---- | ---- | ---- |
| とん | 複合詞（音読み） | 豚汁・豚カツ・豚肉 | 接其他漢字組成料理・食材名 |
| ぶた | 獨立使用（訓読み） | 豚（ぶた） | 指「豬」這個動物本身 |

### 子（こ／し／す）

| 讀音 | 種類 | 例詞 | 說明 |
| --- | ---- | ---- | ---- |
| こ | 訓読み | 子（こ）・子供（こども） | 獨立說「孩子」或日常組合詞 |
| し | 音読み（主） | 原子（げんし）・電子（でんし）・帽子（ぼうし） | 多接漢字，科學／正式詞彙 |
| す | 音読み（副） | 様子（ようす）・椅子（いす） | 少數固定複合詞，不規則殘留 |

⚠️ し和す都是漢語借音，こ是日本原生音；見到子在複合詞裡要注意是走し還是す。

## 日本人姓名讀法

### 五大基本規律

| 規律 | 說明 | 例 |
| --- | --- | --- |
| 1. 姓多用訓読み | 姓氏大多用和語音 | 鈴木（すずき）、淺田（あさだ） |
| 2. **連濁**（れんだく） | 兩字合成，後字清音→濁音 | 田（た）→**だ**、保（ほ）→**ぼ** |
| 3. 藤（ふじ）→ とう | 複合姓後半改音読み | 遠藤（えん**どう**）、佐藤、伊藤 |
| 4. 一字名用訓読み | 單字下名幾乎全是和語音 | 航（わたる）、薫（かおる） |
| 5. 多字名可用音読み | 兩字下名常走漢音 | 大地（**だい**ち） |

### 連濁（れんだく）詳解

> 兩漢字合成詞時，**後字的語頭清音自動濁化**。姓名中最常見：

| 後字單讀 | 姓氏中 | 例 |
| ------- | ------ | -- |
| 田（た） | **だ** | 淺田あさ**だ**・鎌田かま**だ**・梅田うめ**だ**・前田まえ**だ** |
| 保（ほ） | **ぼ** | 久保く**ぼ** |
| 木（き） | き（不濁） | 鈴木すず**き**（例外，部分姓不連濁）|

⚠️ 「田」結尾的姓幾乎 100% 唸だ，是最強規律。
