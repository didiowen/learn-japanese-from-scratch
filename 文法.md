## G1：X は Y です

### 基本結構

```
X  は  Y  です
主語    補語  コピュラ（連繫詞）
```

**は**（發音 wa）= 主題標記，「至於 X 這個主題...」  
**です**（des）= 是（禮貌體）

### 四個變化

| 時態 | 形式 | 意思 |
|------|------|------|
| 現在肯定 | X は Y **です** | X 是 Y |
| 現在否定 | X は Y **じゃありません** | X 不是 Y |
| 疑問 | X は Y **ですか？** | X 是 Y 嗎？ |
| 過去肯定 | X は Y **でした** | X（以前）是 Y |

> **じゃ** 是口語縮略，正式書面用 **では**（ではありません）。

### 例句

| 日文 | 中文 |
|------|------|
| わたしは**がくせい**です。 | 我是學生。 |
| これは**すし**ですか？ | 這是壽司嗎？ |
| あれは**コーヒー**じゃありません。 | 那不是咖啡。 |
| きのうは**にちようび**でした。 | 昨天是星期日。 |

### は 是主題標記，不是等號

は 不是「等於號」，是「現在要說的主角」。  
後面不限定只能接 です，可以接各種述語：

- わたし**は**がくせい**です**（我是學生）
- わたし**は**コーヒー**がすきです**（我喜歡咖啡）
- わたし**は**きょう**いきます**（我今天要去）

### 台語對照

台語的「是」（sī）功能相同，但語序不同：

| 台語 | 日語 |
|------|------|
| 我**是**學生 | わたし**は**がくせい**です** |
| 「是」放中間 | は 標主題，です 放句尾 |

## G2：い形容詞

### 特徵

字典形結尾是 **い**，直接描述名詞或作述語。  
活用時改變 **い** 的部分，其餘不動。

### 四個變化

| 時態 | 語尾變化 | 例（おいしい 好吃） |
|------|---------|----------------|
| 現在肯定 | **～い**です | おいしい**です** |
| 現在否定 | **～くない**です | おいしく**ない**です |
| 過去肯定 | **～かった**です | おいしかった**です** |
| 過去否定 | **～くなかった**です | おいしくなかった**です** |

> 口語常省略 です：おいしい！おいしくない！

### 修飾名詞時

形容詞直接放名詞前，い 不變：

| 日文 | 中文 |
|------|------|
| **たかい**やま | 高的山 |
| **おいしい**すし | 好吃的壽司 |
| **さむい**ひ | 寒冷的日子 |

### 常用い形容詞

| 日文 | 漢字 | 讀法 | 意思 |
|------|------|------|------|
| おおきい | 大きい | ookii | 大 |
| ちいさい | 小さい | chiisai | 小 |
| たかい | 高い | takai | 高／貴 |
| やすい | 安い | yasui | 便宜 |
| あたらしい | 新しい | atarashii | 新 |
| ふるい | 古い | furui | 舊 |
| おいしい | 美味しい | oishii | 好吃 |
| まずい | — | mazui | 難吃 |
| あつい | 暑い／熱い | atsui | 熱（天氣／食物） |
| さむい | 寒い | samui | 冷（天氣） |
| つめたい | 冷たい | tsumetai | 冷（觸感） |
| たのしい | 楽しい | tanoshii | 開心 |
| むずかしい | 難しい | muzukashii | 難 |
| やさしい | 易しい | yasashii | 容易／溫柔 |

> ⚠️ **例外：いい（好）**  
> いい 是不規則，否定／過去要用 **よい** 的活用形：  
> いい → よくないです・よかったです（不說 いくない）

### 台語連結

い形容詞的「くない」否定，概念接近台語「無＋形容詞」：  
おいしく**ない** ≈ **無**好食

## G3：な形容詞

### 特徵

外觀像名詞，修飾名詞時字尾加 **な**（所以叫な形容詞）。  
作述語時用 **です** 連接，活用規則跟 G1 名詞句一樣。

### 四個變化（跟 G1 完全相同）

| 時態 | 形式 | 例（きれい 漂亮） |
|------|------|----------------|
| 現在肯定 | ～**です** | きれい**です** |
| 現在否定 | ～**じゃありません** | きれいじゃありません |
| 過去肯定 | ～**でした** | きれい**でした** |
| 過去否定 | ～**じゃありませんでした** | きれいじゃありませんでした |

### 修飾名詞時加な

| 日文 | 中文 |
|------|------|
| きれい**な**まち | 漂亮的城市 |
| しずか**な**ところ | 安靜的地方 |
| すき**な**たべもの | 喜歡的食物 |

> 作述語時不加な：「まちは**きれいです**」（城市很漂亮）

### 常用な形容詞

| 日文 | 漢字 | 讀法 | 意思 |
|------|------|------|------|
| きれい | 綺麗 | kirei | 漂亮／乾淨 |
| しずか | 静か | shizuka | 安靜 |
| にぎやか | 賑やか | nigiyaka | 熱鬧 |
| べんり | 便利 | benri | 方便 |
| ふべん | 不便 | fuben | 不方便 |
| すき | 好き | suki | 喜歡 |
| きらい | 嫌い | kirai | 討厭 |
| じょうず | 上手 | jouzu | 擅長 |
| へた | 下手 | heta | 不擅長 |
| げんき | 元気 | genki | 有精神 |
| たいせつ | 大切 | taisetsu | 重要 |
| たいへん | 大変 | taihen | 辛苦／很（程度副詞） |
| ゆうめい | 有名 | yuumei | 有名 |
| ひま | 暇 | hima | 空閒 |

### い形容詞 vs な形容詞 快速辨別

い結尾不一定是い形容詞！這幾個是**な形容詞**（要特別記）：

| 容易誤判的な形容詞 | 意思 |
|-----------------|------|
| きれい（綺麗）**い** | 漂亮 |
| きらい（嫌い）**い** | 討厭 |
| ゆうめい（有名）**い** | 有名 |

記憶訣：這些字的 い 是漢字的一部分，不是活用語尾。

## G4：あります／います

### 功能

表示「存在」——東西在哪裡、有沒有某東西。

| 動詞 | 用於 | 意思 |
|------|------|------|
| **あります** | 無生命的物（物品、店、事件） | 有／在 |
| **います** | 有生命的（人、動物） | 有／在 |

> ⚠️ 植物用 あります，機器人用 あります，但機器人如果像人就用 います（語感問題）。

### 句型一：X が あります／います（有 X）

| 日文 | 中文 |
|------|------|
| ねこ**が** います。 | 有貓。 |
| コンビニ**が** あります。 | 有便利商店。 |
| じかん**が** ありません。 | 沒有時間。 |

### 句型二：X は Y に あります／います（X 在 Y）

```
X  は  場所  に  あります／います
主語     地點  助詞
```

| 日文 | 中文 |
|------|------|
| ねこは**テーブルの上**にいます。 | 貓在桌子上。 |
| コンビニは**えきの前**にあります。 | 便利商店在車站前。 |
| わたしは**うち**にいます。 | 我在家。 |

### 位置詞小結

| 日文 | 讀法 | 意思 |
|------|------|------|
| うえ | ue | 上面 |
| した | shita | 下面 |
| まえ | mae | 前面 |
| うしろ | ushiro | 後面 |
| となり | tonari | 旁邊（隔壁） |
| よこ | yoko | 旁邊（側邊） |
| なか | naka | 裡面 |
| そと | soto | 外面 |
| あいだ | aida | 中間 |

使用時加 **の**：テーブル**の**うえ（桌子的上面）

### あります 的特殊用法：事件

| 日文 | 中文 |
|------|------|
| あした**テスト**があります。 | 明天有考試。 |
| らいしゅう**パーティー**があります。 | 下週有派對。 |

## G5：ます形（動詞禮貌體）

日語動詞接上 ます 系列自動變禮貌體，變化規則完全固定。

| 語尾 | 時態／語氣 | 例子 |
|------|---------|------|
| ます | 現在肯定 | わかります・ございます |
| ません | 現在否定 | わかりません |
| ました | 過去肯定 | わかりました・ございました |
| ませんでした | 過去否定 | わかりませんでした |

**ありがとうございます vs ございました**

- ございます：當下發生的事（店員現在幫你服務）
- ございました：事情已結束（離開餐廳，感謝整頓飯）

### る動詞（一段動詞）

以る結尾，直接套ます系列：

| 原形 | 禮貌體 | 否定 | 過去 |
|------|--------|------|------|
| たべる（吃） | たべます | たべません | たべました |
| みる（看） | みます | みません | みました |

### する動詞

名詞＋する，把名詞變成動詞：

| 名詞 | ＋する | 禮貌體 |
|------|--------|--------|
| べんきょう | べんきょうする | べんきょうします |
| うんどう | うんどうする | うんどうします |
| りょこう | りょこうする | りょこうします |
| でんわ | でんわする | でんわします |

> する → します（禮貌）・しません（否定）・しました（過去）

## 助詞

| 助詞 | 發音 | 功能 | 例子 |
|------|------|------|------|
| は | wa | 主題標記 | ぼく**は** |
| を | o | 受詞標記（動作對象） | ごはん**を**たべる |
| の | no | 所有／修飾（「的」） | にほん**の**りょうり |
| に | ni | 方向／對象／位置 | にほん**に**いく |
| が | ga | 主語標記（對比 は） | ねこ**が**います |
| ね | ne | 尋求共鳴（「對吧？」） | そうです**ね** |
| か | ka | 疑問句結尾 | そうです**か** |

## 接續詞

| 接續詞 | 意思 | 語感 |
|--------|------|------|
| だから | 所以 | 解釋原因，有時帶不耐煩 |
| でも | 但是 | 轉折，口語常用 |
| そして | 然後／而且 | 連接，較正式 |

## 名字後綴（敬稱）

| 後綴 | 漢字 | 語感 | 用於 |
|------|------|------|------|
| さん | 様（簡化） | 禮貌正式 | 任何人 |
| くん | 君 | 輕鬆、上對下 | 男性晚輩／同輩 |
| ちゃん | —（從さん演變） | 親暱撒嬌 | 小孩、親近的人、可愛事物 |
| さま | 様 | 非常正式尊敬 | 客人、神明、信件 |

ちゃん 是小孩把さん說不準演變來的，天生帶著「可愛、不正式」的語感。おねえちゃん = 姊姊的撒嬌暱稱版。
