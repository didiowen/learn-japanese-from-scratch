# 日文學習筆記與測驗

**從台語記憶日文・五十音學習系統**

這個 repo 是一套用 Claude AI 輔助建立的日文自學系統，起點是一份「本來就會的單字片語」清單，透過 Claude Chat 教平假名、一邊學一邊把工具建起來。

## 這套系統是什麼

- **互動測驗**：平假名、片假名各有獨立的 SRS 記憶測驗（間隔複習，答對拉長間隔、答錯重置）
- **學習筆記**：用 Docsify 渲染的 Markdown 筆記站，記錄字形來源、記憶鉤、發音規則
- **台語記憶優勢**：日語音読み和台語文言音同源，超過 1000 個對應詞——大量記憶鉤直接用台語、注音或粵語

## 學習方式

1. 把自己本來就會的單字填入 `already-known.md`，測驗中這些字的出現頻率會自動降低
2. 請 Claude Chat 教假名：**每次 10–15 個以內**，記熟了再繼續下一批
3. 每個假名都有漢字來源 ＋ 記憶鉤，Claude 會根據你的母語（台語／華語／粵語）選最近的聯想
4. 新字學完後 Claude 會更新測驗和筆記，下次打開就能練習

## 測驗功能

| 功能 | 說明 |
|------|------|
| 字→音 選擇題 | 看片假名選讀音 |
| 字→音 輸入 | 直接打羅馬字 |
| 音→字 選擇題 | 聽到讀音選字形 |
| SRS 間隔複習 | 答對升級、拉長複習間隔 |
| 單字模式 | 外來語單字卡，可依主題篩選 |

## Fork 來用

### 1. Fork 這個 repo

GitHub 右上角按 **Fork**，建立你自己的副本。

### 2. 修改 `already-known.md`

把你本來就認識的日文單字、片語填進去（格式照現有的表格）。測驗系統會讓這些字從第二輪才開始密集出現。

### 3. 用 Claude Code（或 Claude Chat）開始學

在自己的 repo 裡打開 Claude Code（或把檔案貼給 Claude Chat），說：

> 「我想學平假名，請從我不認識的開始，一次 10 個。」

`CLAUDE.md` 裡記錄了這套系統的所有規則——記憶鉤優先順序、發音九條規律、更新測驗的方式，Claude 讀完就知道怎麼配合你學習。

### 4. 學習流程

```
學新假名 → Claude 更新筆記 + 測驗
  ↓
用測驗練習到記熟
  ↓
繼續下一批
```

### 5. 部署（可選）

直接用 GitHub Pages 部署靜態網站，`index.html` 是入口頁，`notes/` 是 Docsify 筆記站。

## 檔案結構

```
├── index.html              # 入口頁（測驗＋筆記連結）
├── hiragana-quiz.html      # 平假名 SRS 測驗
├── katakana-quiz.html      # 片假名 SRS 測驗
├── notes/                  # Docsify 筆記站
├── hiragana.md             # 平假名字表、記憶鉤、語法筆記
├── katakana.md             # 片假名字表（按學習輪次整理）
├── vocabulary.md           # 單字表（依主題分類）
├── grammar.md              # N5 語法筆記
├── conversation.md         # 會話筆記
├── kanji.md                # 漢字筆記（音讀み＋台語對照）
├── language-notes.md       # 台語／粵語／注音記憶鉤對照表
├── already-known.md        # 已知單字，測驗中降低出現頻率
└── CLAUDE.md               # Claude 行為指南（學習規則、更新流程）
```

## 適合誰用

- 母語台語或華語、想找記憶鉤快速記假名的人
- 想要 SRS 測驗但不想用 Anki 的人
- 想讓 AI 陪著學、邊學邊把工具建起來的人
