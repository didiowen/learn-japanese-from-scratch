# 日文學習筆記與測驗

**從台語記憶日文・五十音學習系統**

這個 repo 是一套用 Claude AI 輔助建立的日文自學系統，起點是一份「本來就會念的單字片語」清單，透過 Claude Chat 以這些熟悉的詞為跳板教平假名，一邊學一邊把工具建起來。

## 核心學習邏輯

`already-known.md` 裡收集的是**日常生活中聽過、會念，但不會讀寫**的日文詞——日本料理名稱、常聽到的片語、地名等等。

Claude 拿這份清單當跳板：「你已經知道 sakura 怎麼念，現在來學 さ・く・ら 三個字怎麼寫。」**從你熟悉的聲音出發，學字形**，比從零開始快很多。

學會假名之後，這些字也自然就能讀和寫了。

## 這套系統有什麼

- **互動測驗**：平假名、片假名各有獨立的 SRS 記憶測驗（答對拉長複習間隔、答錯重置）
- **學習筆記**：用 Docsify 渲染的 Markdown 筆記站，記錄字形來源、記憶鉤、發音規則
- **台語記憶優勢**：日語音読み和台語文言音同源，超過 1000 個對應詞——大量記憶鉤直接用台語、注音或粵語

## Fork 來用

### 1. Fork 這個 repo

GitHub 右上角按 **Fork**，建立你自己的副本。

### 2. 整理你的 `already-known.md`

把你平常聽過、會念但不會寫的日文詞填進去（格式照現有的表格）。這份清單是你的起點，也是 Claude 設計第一堂課的材料。

範例：你知道「壽司」唸 sushi、「一」唸 ichi、「先生」唸 sensei——把這些填進去，Claude 就可以從這些詞裡出現的假名開始教。

### 3. 請 Claude Chat 開始教

把 `already-known.md` 的內容貼給 Claude Chat，說：

> 「這是我日常生活中聽過會念但不會讀寫的日文，請根據這些字出現的平假名來教我，一次 10–15 個以內。」

Claude 會從你熟悉的詞挑出最常見的假名先教，每個字給漢字來源和記憶鉤，學完再繼續下一批。

### 4. 學習流程

```
填入你熟悉的詞 (already-known.md)
  ↓
Claude 從這些詞的假名開始教（每次 10–15 個）
  ↓
用測驗練習到記熟
  ↓
繼續下一批，學完後那些詞你也能讀和寫了
```

### 5. 用 Claude Code 維護（可選）

如果你有 Claude Code，可以在 repo 裡直接開 session——`CLAUDE.md` 記錄了這套系統的所有規則，Claude 讀完就知道怎麼更新測驗和筆記。

### 6. 部署（可選）

直接用 GitHub Pages 部署，`index.html` 是入口，`notes/` 是 Docsify 筆記站。

## 學習效果最好的方式

- **每次 10–15 個假名**，記熟再繼續，不要一次學完整行
- 用你本來就會念的詞當錨點，看到字形就能想起聲音
- 善用台語記憶鉤——台語和日語共享大量中古漢語音，很多假名的漢字來源直接用台語念就夠了

## 檔案結構

```
├── index.html              # 入口頁（測驗＋筆記連結）
├── hiragana-quiz.html      # 平假名 SRS 測驗
├── katakana-quiz.html      # 片假名 SRS 測驗
├── notes/                  # Docsify 筆記站
├── hiragana.md             # 平假名字表、記憶鉤、語法筆記
├── katakana.md             # 片假名字表（按學習輪次整理）
├── vocabulary.md           # 單字表（依主題分類）
├── grammar.md              # N5 語法筆記
├── conversation.md         # 會話筆記
├── kanji.md                # 漢字筆記（音読み＋台語對照）
├── language-notes.md       # 台語／粵語／注音記憶鉤對照表
├── already-known.md        # 會念但不會讀寫的詞，是學習起點
└── CLAUDE.md               # Claude 行為指南（學習規則、更新流程）
```

## 適合誰用

- 有日文聽力基礎（從日劇、日語歌、日本食物）、想把「會念」變成「會讀寫」的人
- 母語台語或華語、想找記憶鉤快速記假名的人
- 想讓 AI 陪著學、邊學邊把工具建起來的人
