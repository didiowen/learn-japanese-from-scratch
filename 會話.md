# 會話

- Ni-hon-go, cho-tto...：日文（我懂）一點點而已...
- **Nihongo ga zenzen wakarimasen**：我完全不會日文
- **Mou-ichi-do, one-gai-shi-mas-su**：再說一次 
- **場合：** 真的沒聽清楚，希望對方重複一遍時

### 請講慢一點 **Yukkuri, one-gai-shi-mas-su**
- **場合：** 對方語速太快，你希望他一字一字慢慢說時

當你遇到店員講了一串長長的日文，而你完全當機時，請按照這個順序組合：

1. **先道歉：** **Sumimasen.** (速依馬線，不好意思)    
2. **提要求：** **Yukkuri, one-gai-shi-mas-s.** (優哭力... 麻煩講慢一點)
3. **如果還是不懂：** **Wakarimasen.** (哇嘎力馬線，我不懂)

## 購物

### 袋子
- **Fukuro** = 袋子
- **Reji-bukuro** = 購物袋（收銀機袋）
- **Ba-gu**
- **Bi-ni-ru-ba-gu**＝ **Vinyl Bag**（塑膠袋）

- **要：** **Hai, one-gai-shi-masu.**  —— 「對，麻煩了。」
- **不要：** **I-i-e, dai-jo-bu desu.**  —— 「不用，沒關係。」

### 這個，請給我：[指著東西] + **Kore, o-ne-gai-shi-masu**
- **場合：** 買東西、餐廳點餐指著菜單圖片

### 有...嗎？：[東西] + **a-ri-ma-su-ka?**
- **場合：** 藥妝店找藥、超市找牛奶。    
    - _Men-tai-ko（明太子）阿力馬斯嘎？_

### 「多少錢？」**I-ku-ra de-s-ka?**
- **用法：** 指著沒標價的東西，或是結帳時確認總金額

### 「可以刷卡嗎？」**Ka-do de-s-ka?** 
- **用法：** 結帳時拿出信用卡示意。雖然文法簡略，但店員絕對懂你在問「Card Okay?」

### 「這是什麼？」**Ko-re wa nan de-s-ka?**
- **用法：** 看到奇怪的御飯糰口味或是紀念品時

### 有免稅嗎？**Tax-free, de-s-ka?**

### 有新的嗎？**A-ta-ra-shi-i, a-ri-mas-ka?**  
- 想要拿新的（不想拿架上展示品）

### 試穿（可以）嗎？**Shi-cha-ku, de-s-ka?**


## 點餐

Irasshaimase（歡迎光臨）

請問幾位？Nan-mei-sama de-s-ka? 
- 一位：**Hi-to-ri de-s**
- 兩位：**Fu-ta-ri de-s**
- 三位：**San-nin de-s**

| **禮貌程度**    | **你的說法 (擬音)**              | **適合什麼時候用**                 |
| ----------- | -------------------------- | --------------------------- |
| **稍微有禮貌**   | **Chotto-matte-ne**        | 店員只是要跟你拿個東西。                |
| **正式有禮貌**   | **Chotto-matte-kuda-sa-i** | 你在翻錢包找零錢時。                  |
| **最道地(推薦)** | **Ma-da de-s** (馬打 ㄉㄟˇ絲)   | **「還沒(好)」**。店員問你點餐了嗎？你就說這句。 |

當你點完餐後，店員會唸一次所有菜名，最後會問：**I-jo de yo-ro-shi-i de-s-ka?** （以上點餐可以嗎？）
 - **Hai, i-jo de-s.** —— 對，就這些。

**Kore-de** 
- **意思：** 就選這個。
- **用法：** 指著你要的東西，後面加個 `de`，店員就知道你決定好了。

**日本有些餐廳進門要「脫鞋」。** 如果你看到店門口有鞋櫃，或者店員指著地板說 **「Ku-tsu...」** (苦粗)，那就是要脫鞋的意思。

| **想要什麼** | **你的說法 (擬音)**                         | **意思**  |
| -------- | ------------------------------------- | ------- |
| **菜單**   | **Me-nyu, one-gai-shi-mas-s**         | 請給我菜單。  |
| **開水**   | **O-mi-zu, one-gai-shi-mas-s**        | 請給我水。   |
| **濕紙巾**  | **O-shi-bo-ri, one-gai-shi-mas-s**    | 請給我濕紙巾。 |
| **中文菜單** | **Chu-go-ku-go me-nyu, a-ri-mas-ka?** | 有中文菜單嗎？ |
  
### 多謝款待 **Go-chi-so-sa-ma de-shi-ta**
- 離店說

| **意思**     | **關東 (東京常用)** | **關西 (大阪常用)** | **擬音與感覺**                  |
| ---------- | ------------- | ------------- | -------------------------- |
| **真的嗎？**   | **Hon-to?**   | **Hon-ma?**   | **烘馬？** (關西人幾乎不說 Hon-to)   |
| **不行/不可以** | **Da-me**     | **A-kan**     | **阿亢** (在大阪看到小孩亂跑，爸媽會喊阿亢！) |
| **謝謝**     | **Arigato**   | **O-ki-ni**   | **歐key尼** (在京都或老店比較常聽到)    |
