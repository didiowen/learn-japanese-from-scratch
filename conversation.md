## 語言障礙求生包

- **Nihongo, chotto...** — 日文（我懂）一點點而已
- **Nihongo ga zenzen wakarimasen** — 我完全不會日文
- **Mou ichido, onegaishimasu** — 再說一次
- **Yukkuri, onegaishimasu** — 請講慢一點

當你遇到店員講了一串長長的日文而完全當機時，按順序組合：

1. **先道歉**：すみません
2. **提要求**：ゆっくり、おねがいします
3. **還是不懂**：わかりません

## 購物

### 袋子

- ふくろ＝袋子
- レジぶくろ＝購物袋
- ビニールバッグ＝塑膠袋

- **要**：はい、おねがいします

## 點餐

**歡迎光臨**：いらっしゃいませ

**請問幾位**：なんめいさまですか？

| 人數 | 說法        |
| --- | ---------- |
| 一位 | ひとりです    |
| 兩位 | ふたりです    |
| 三位 | さんにんです   |

**等一下的說法**：

| 禮貌程度    | 說法                  | 適合時機              |
| --------- | -------------------- | ------------------- |
| 稍微有禮貌   | ちょっとまってね            | 店員要跟你拿個東西（て形口語） |
| 正式有禮貌   | ちょっとまってください         | 翻錢包找零錢時（てください → G9） |
| 最道地（推薦） | まだです                | 還沒點好，店員來問時        |

**點餐結束確認**：いじょうでよろしいですか？（以上可以嗎？）
- 回答：はい、いじょうです

**就選這個**：これで（指著 ＋ de，店員就懂你決定了）

**餐廳常用**：

| 想要什麼 | 說法                          |
| ------- | ----------------------------- |
| 菜單    | メニュー、おねがいします            |
| 開水    | おみず、おねがいします             |
| 濕紙巾   | おしぼり、おねがいします            |
| 中文菜單  | ちゅうごくごメニュー、ありますか？     |

**離開時**：ごちそうさまでした（多謝款待）
