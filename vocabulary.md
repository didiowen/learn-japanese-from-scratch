### 名字後綴（敬稱）

| 後綴 | 漢字 | 語感 | 用於 |
| ---- | --------- | -------- | ------------ |
| さん | 様（簡化） | 禮貌正式 | 任何人 |
| くん | 君 | 輕鬆、上對下 | 男性晚輩／同輩 |
| ちゃん | —（從さん演變）| 親暱撒嬌 | 小孩、親近的人、可愛事物|
| さま | 様 | 非常正式尊敬 | 客人、神明、信件 |

ちゃん 是小孩把さん說不準演變來的，天生帶著「可愛、不正式」的語感。おねえちゃん = 姊姊的撒嬌暱稱版。

### 百・千的複合數

**百（ひゃく）**

|數字|讀法|變化說明|
|---|---|---|
|100|ひゃく|基本形|
|200|にひゃく||
|300|さんびゃく|h → b（濁化）|
|400|よんひゃく||
|500|ごひゃく||
|600|ろっぴゃく|く脫落變促音＋h → p（半濁）|
|700|ななひゃく||
|800|はっぴゃく|ち脫落變促音＋h → p（半濁）|
|900|きゅうひゃく||

**千（せん）**

|數字|讀法|變化說明|
|---|---|---|
|1000|せん|基本形（不說いちせん）|
|2000|にせん||
|3000|さんぜん|s → z（濁化）|
|4000|よんせん||
|5000|ごせん||
|6000|ろくせん||
|7000|ななせん||
|8000|はっせん|ち脫落變促音|
|9000|きゅうせん||

規律整理：300／600／800 的百位，3000／8000 的千位出現變化。600 和 800 是 **促音＋半濁**，300 和 3000 是**濁化**，8000 只有**促音**。
